AWS (Amazon Web Services) offers a wide array of cloud services across various domains, from compute and storage to machine learning, databases, and networking. Here's a comprehensive list of the most common services grouped by category:

### **1. Compute Services**
- **EC2 (Elastic Compute Cloud):** Scalable virtual servers in the cloud.
- **Lambda:** Serverless compute service to run code without provisioning servers.
- **ECS (Elastic Container Service):** Managed container service for running Docker containers.
- **EKS (Elastic Kubernetes Service):** Managed Kubernetes service.
- **Elastic Beanstalk:** Platform as a service (PaaS) for deploying web apps and services.
- **Fargate:** Serverless compute for containers.
- **Lightsail:** Simplified virtual servers, databases, and storage.
- **Outposts:** Extend AWS infrastructure to on-premises environments.

### **2. Storage Services**
- **S3 (Simple Storage Service):** Scalable object storage for various types of data.
- **EBS (Elastic Block Store):** Block storage for EC2 instances.
- **EFS (Elastic File System):** Scalable file storage for EC2 instances.
- **FSx for Windows/Lustre:** Managed Windows File Servers and high-performance file systems.
- **S3 Glacier:** Low-cost, long-term cold storage.
- **Storage Gateway:** Hybrid storage integration between on-premises and AWS.

### **3. Database Services**
- **RDS (Relational Database Service):** Managed relational databases like MySQL, PostgreSQL, Oracle, SQL Server, and MariaDB.
- **DynamoDB:** Fully managed NoSQL database.
- **Aurora:** High-performance relational database compatible with MySQL and PostgreSQL.
- **Redshift:** Data warehousing service.
- **Neptune:** Fully managed graph database service.
- **ElastiCache:** Managed in-memory data stores compatible with Redis and Memcached.
- **DMS (Database Migration Service):** Easily migrate databases to AWS.

### **4. Networking & Content Delivery**
- **VPC (Virtual Private Cloud):** Isolated network environment for your AWS resources.
- **Route 53:** Scalable DNS and domain name management service.
- **CloudFront:** Global content delivery network (CDN).
- **Direct Connect:** Dedicated network connection from your premises to AWS.
- **Global Accelerator:** Improves availability and performance of applications with global users.
- **Elastic Load Balancing (ELB):** Distributes incoming traffic across multiple targets.
- **API Gateway:** Managed service to create, publish, and monitor APIs.

### **5. Security, Identity & Compliance**
- **IAM (Identity and Access Management):** Manage access to AWS resources.
- **Cognito:** User identity and authentication service for web and mobile apps.
- **KMS (Key Management Service):** Manage encryption keys.
- **CloudHSM:** Hardware-based key management for regulatory compliance.
- **GuardDuty:** Continuous threat detection service.
- **Inspector:** Automated security assessment for applications.
- **Macie:** Machine learning-powered security service to protect data in S3.
- **WAF (Web Application Firewall):** Protect web applications from common web exploits.
- **Secrets Manager:** Manage and retrieve secrets securely.
- **Certificate Manager (ACM):** Provision and manage SSL/TLS certificates.

### **6. Machine Learning**
- **SageMaker:** Build, train, and deploy machine learning models at scale.
- **Rekognition:** Image and video analysis service.
- **Comprehend:** Natural Language Processing (NLP) service for text analysis.
- **Lex:** Build conversational interfaces (chatbots).
- **Polly:** Text-to-speech service.
- **Translate:** Language translation service.
- **Transcribe:** Speech-to-text service.
- **Personalize:** Real-time personalization and recommendation service.
- **DeepLens:** Machine learning-enabled video camera for developers.
- **Forecast:** Forecasting models based on time-series data.
- **Fraud Detector:** Automatically detect fraud in real-time.

### **7. Analytics**
- **Athena:** Query data stored in S3 using SQL.
- **EMR (Elastic MapReduce):** Managed Hadoop and Spark clusters.
- **Kinesis:** Real-time data streaming and analytics.
- **QuickSight:** Business intelligence service for data visualization.
- **Data Pipeline:** Orchestration service to move and process data.
- **Glue:** Managed ETL (Extract, Transform, Load) service.
- **Lake Formation:** Build, secure, and manage data lakes.
- **OpenSearch (formerly Elasticsearch Service):** Managed search and analytics engine.
- **Redshift Spectrum:** Run queries against S3 data without loading it into Redshift.

### **8. Developer Tools**
- **CodeCommit:** Source control service similar to Git.
- **CodeBuild:** Build and test code in the cloud.
- **CodeDeploy:** Automate code deployments.
- **CodePipeline:** Automate CI/CD pipelines.
- **Cloud9:** Cloud-based integrated development environment (IDE).
- **X-Ray:** Analyze and debug distributed applications.
- **AWS SAM (Serverless Application Model):** Framework to build serverless applications.
  
### **9. Internet of Things (IoT)**
- **IoT Core:** Connect IoT devices to the AWS cloud.
- **Greengrass:** Run local compute, messaging, and data caching for connected devices.
- **IoT Analytics:** Analyze IoT device data.
- **FreeRTOS:** Operating system for microcontrollers.
- **IoT Device Defender:** Secure your IoT fleet.
- **IoT SiteWise:** Collect, analyze, and visualize industrial equipment data.

### **10. Migration & Transfer**
- **DMS (Database Migration Service):** Migrate databases to AWS.
- **Snowball:** Transfer large amounts of data using physical storage devices.
- **Transfer Family:** Fully managed SFTP, FTPS, and FTP service.
- **Server Migration Service (SMS):** Migrate on-premises servers to AWS.
- **Migration Hub:** Centralized location to track migrations across services.
  
### **11. Customer Engagement**
- **Connect:** Cloud-based contact center service.
- **Pinpoint:** Marketing campaigns and customer engagement.
- **SES (Simple Email Service):** Send and receive emails.
- **SNS (Simple Notification Service):** Pub/Sub messaging service for notifications.
- **SQS (Simple Queue Service):** Fully managed message queuing service.

### **12. Management & Governance**
- **CloudWatch:** Monitoring and observability of AWS resources and applications.
- **CloudTrail:** Log and track AWS API calls.
- **Config:** Track AWS resource configurations and compliance.
- **OpsWorks:** Configuration management service supporting Chef and Puppet.
- **Systems Manager:** Centralized resource management for operational tasks.
- **Service Catalog:** Create and manage catalogs of IT services.
- **Trusted Advisor:** Recommendations to optimize AWS performance and security.
  
### **13. Media Services**
- **Elastic Transcoder:** Convert media files to different formats.
- **MediaConvert:** File-based video transcoding.
- **MediaLive:** Broadcast-grade live video streaming.
- **MediaPackage:** Deliver streaming content to end-users.
- **MediaStore:** Store and deliver media assets.
- **MediaTailor:** Video personalization and monetization service.

### **14. Blockchain**
- **Managed Blockchain:** Create and manage blockchain networks using Hyperledger Fabric or Ethereum.

### **15. Quantum Computing**
- **Braket:** Develop and test quantum algorithms on simulated or actual quantum computers.

### **16. Satellite**
- **Ground Station:** Satellite ground station as a service to control satellite communications.

---

These services cater to different aspects of cloud computing, including compute, storage, databases, analytics, security, machine learning, and more, making AWS a comprehensive platform for building and scaling applications.
